## Linear Regression

$$
h(x) = x^T \hat{w}
$$

$$
\hat{w}=\operatorname{argmin}_{w} \sum_{x, y}\left(y-x^{T} w\right)^{2}
$$

$$
\hat { w }^ = \left( \gamma \mathbf { I } + X ^ { T } X \right) ^ { - 1 } X ^ { T } y
$$
