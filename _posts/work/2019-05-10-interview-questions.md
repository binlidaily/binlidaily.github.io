### 1.1 逻辑斯特回归(LR)
1. LR 推导（伯努利过程，极大似然，损失函数，梯度下降）有没有最优解？
  
  * 假设 LR 符合伯努利分布，通过 GLM 能得到决策函数是 Sigmoid 函数，然后最大似然估计流程走一套就能得到交叉熵损失，优化则求导然后用梯度下降。
  
  * 逻辑斯特回归是凸优化问题，凸函数+无约束条件，有最优解，因为存在一阶导数为零的点。
  
2. LR 可以用核么？可以怎么用？L1 和 L2 正则项是啥？LR 加 L1 还是 L2 好？加哪个可以用核？

   * 有核逻辑斯特回归。
   * 加 L2 正则项，和 svm 类似，加 L2 正则项可以用核方便处理

3. LR 跟一层神经网络的区别？

   * 区别在于 LR 只能做二分类，神经元可以做多分类。

4. [LR 可以用来处理非线性问题么？](https://www.zhihu.com/question/29385169)怎么做？可以像SVM那样么？为什么？

   * 可以使用核逻辑斯特回归来解决非线性问题，本质就是调用 LR，不过加核是显式地把特征映射到高维 然后再做 LR。
   * 对特征做非线性变换，比如kernel，最后一层看成是 LR，前面看成是提特征。
   * LR 的应用场景主要是特征很多的情况

5. 为什么 LR 需要归一化或者取对数，为什么 LR 把特征离散化后效果更好，为什么把特征组合之后还能提升？
   * 归一化：可以提高收敛速度，提高收敛的精度；
   * 反正这些基本都是增强了特征的表达能力，或者说更容易线性可分吧

6. [SVM 和 LR 分别在什么情况下使用？](https://www.zhihu.com/question/21704547)

   * 同样的线性分类情况下，如果异常点较多的话，无法剔除，首先 LR，LR 中每个样本都是有贡献的，最大似然后会自动压制异常的贡献，SVM+软间隔对异常还是比较敏感，因为其训练只需要支持向量，有效样本本来就不多，一旦被干扰，预测结果难以预料。
   * LR 用 Kernel 容易过拟合，SVM 不容易过拟合。

14. LR 为什么用 Sigmoid函数。这个函数有什么优点和缺点？为什么不用其他函数？

    * GLM 推出来的
    * 优点：
      1. Sigmoid 函数的输出在 $(0, 1)$ 之间，输出范围有限，优化稳定，可以用作输出层。
      2. 连续函数，便于求导。
    * 缺点：
      1. Sigmoid  函数在变量取绝对值非常大的正值或负值时会出现饱和现象，意味着函数会变得很平，并且对输入的微小改变会变得不敏感。
      2. 在反向传播时，当梯度接近于0，权重基本不会更新，很容易就会出现梯度消失的情况，从而无法完成深层网络的训练。
      3. Sigmoid 函数的输出不是 0 均值的，会导致后层的神经元的输入是非 0 均值的信号，这会对梯度产生影响。
      4. 计算复杂度高，因为 Sigmoid 函数是指数形式。
