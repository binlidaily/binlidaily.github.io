In MySQL, the meaning of LIMIT n1, n2 is :

```
n1 : starting index
n2 : number of record/data you want to show
```

For example :

```
ID
-------------------------
1  ------------ > index 0
2
3
4
5
6
7
8
9
10 ------------ > index 9
```

Now if you write query like

```
SELECT * from tbl_name LIMIT 0,5
Output : 
1
2
3
4
5
```
