　　特别注意 MacOS 上启动 MySQL 服务方式的问题：
* 运行 `brew service start mysql` 可以后台启动 mysql
* 运行 `mysql.server start` 前台启动 mysql
    * `mysql.server stop`

登陆 MySQL：
```python
mysql -u root -p
```
