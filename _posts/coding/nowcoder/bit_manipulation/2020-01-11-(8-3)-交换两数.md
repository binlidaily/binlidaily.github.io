## Description

请编写一个算法，不用任何额外变量交换两个整数的值。

给定一个数组**num**，其中包含两个值，请不用任何额外变量交换这两个值，并将交换后的数组返回。

测试样例：

```
[1,2]
返回：[2,1]
```


## Solutions
### 1. Bit Manipulation

```python
# -*- coding:utf-8 -*-

class Swap:
    def getSwap(self, num):
        num[0] = num[0] ^ num[1]
        num[1] = num[0] ^ num[1]  # num[1] = num[0] ^ num[1] ^ num[1], b = a
        num[0] = num[0] ^ num[1]  # num[0] = num[0] ^ num[1] ^ num[0] ^ num[1] ^ num[1], a = b
        return num
```
